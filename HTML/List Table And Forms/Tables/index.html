Tables

  • table tag is used to create tables
  • Tags in tables
    1. <tr> Table Row Used to define a row in HTML table
    2. <td> Table Data Used to define actual data in HTML table
    3. <th> Table Heading Used for header cells within a row.
      Text is bold andcentered by default
  • Caption:
    • Purpose:It provides little description for table
    • Placement:Must be inserted immediately after < table > opening
    • Allignment:Centered above the table by default
