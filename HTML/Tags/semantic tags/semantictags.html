Semantic and Non Semantic Tag

Semantic Tags

    Semantic tags are used to describe the content
  1. Meaningful:Describe Content
  2. SEO:Good For Search Engines
  3. Accessibility: Useful for screen readers
  4. Examples
    1. <header>........</header>
    2. <footer>........</footer>
    3. <article>........</article>
    4. <section>........<section>
    5. <nav>........<nav>

Non Semantic tags

  1. Generic:No Specific Meaning
  2. For Styling:Used For Layout
  3. No SEO:Not SEO friendly
  4. Examples:
    • <div>.......</div>
    • <p>.......</p>
    • <i>.......</i>
