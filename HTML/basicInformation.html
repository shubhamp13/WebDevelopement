Important Points

  • HTML stands for Hyper Text Markup Language
  • HTMl is structure of web page
  • CSS is color and styling of the web page
  • JavaScript make our web page responsive and interactive
  • index.html file is root file for project server will find index.html file while loading.If we dont mention any file in url then automatically server will find to load index.html.it is like home page of our project through which we can go anywhere
  • HTML is case insensitive language
