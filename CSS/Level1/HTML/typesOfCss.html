Types Of CSS

  1. Inline Css
    • It is used to apply style directly to the HTML element with the Style Attribute
    • Ideal for single element styling


  2. Internal CSS
    • Embeded CSS: Style are placed within <style> tags in the HTML head section
    • Cleaner than Inline: More Orgnized compare to Inline style
    • Resuable code


  3. External Css:
    • We can create .css file to include css
    • We can include that css file with < link rel= "stylesheet" href="style.css">

Priority Of CSS

  1. Inline css (Most Priority)
  2. Internal Css (Second Priority)
  3. External Csss (Least Priority)
