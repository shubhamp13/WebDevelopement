Id And Classes

Id Selector

  1. Unique Identifier:Targets a Specific element by Unique id attribute
  2. Syntax:It uses the # symbol
  3. id must be unique per web page
  4. ideal for styling individual distinct elements


Class Selector

  1. Group Styling:Allows Styling for multiple elements grouped under one class
  2. Syntax:It uses (.)symbol
  3. It can bve used for styling multiple elements under one class
